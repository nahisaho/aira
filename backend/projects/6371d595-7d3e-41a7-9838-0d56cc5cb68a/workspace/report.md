# 次世代固体電解質探索 — 総合レポート

## Microsoft+PNNL N2116を超える固体電解質候補の仮説提案

**プロジェクト日付**: 2026-05-03  
**対象分野**: エネルギー材料（全固体電池）  
**目標**: イオン伝導度 σ > 10⁻³ S/cm（室温）を達成する新規固体電解質組成の提案

---

## 1. エグゼクティブサマリー

Microsoft + PNNL が3,260万候補から発見した固体電解質 N2116（Li+Na系、Li使用量70%削減）は画期的だが、イオン伝導度は ≈5×10⁻⁴ S/cm と実用水準（10⁻³ S/cm 以上）に未到達である。本研究では、35本の主要文献の横断調査と知識グラフ分析に基づき、N2116を超える3つの次世代固体電解質候補を仮説として提案する。

### 主要成果

| 仮説 | 組成 | 予測伝導度 | 構造 | 核心戦略 |
|------|------|-----------|------|----------|
| H1 | Li₂.₅(Ti,Zr,Hf,Sn)₀.₅O₀.₅(Cl,Br)₂.₅ | 5×10⁻³ S/cm | 高エントロピーオキシハライド | エントロピー安定化 + 混合アニオン |
| H2 | Li₆.₅P₀.₅Si₀.₅S₅Cl₀.₅F₀.₅ | 1.2×10⁻² S/cm | 変形アルジロダイト | F⁻導入 + Li過剰化 |
| H3 | Li₂.₅Na₀.₅O(Cl₀.₇Br₀.₃) | 3.5×10⁻³ S/cm | 混合アルカリ・アンチペロブスカイト | N2116概念の構造転写 |

## 2. 文献調査結果

### 2.1 調査方法

固体電解質に関する主要論文35本を横断検索し、以下のカテゴリで体系化した：

- **硫化物系** (LGPS, Li₇P₃S₁₁, argyrodite): 最高伝導度 2.5×10⁻² S/cm
- **ハライド系** (Li₃YCl₆, Li₃InCl₆, Li₃ScCl₆): 急速に進展中、5×10⁻³ S/cm 到達
- **酸化物系** (LLZO, NASICON, LATP): 安定性は高いが伝導度に限界
- **オキシハライド系**: 酸化物の安定性とハライドの高伝導度を両立
- **高エントロピー系**: 2024年以降急速に台頭、5.2×10⁻³ S/cm 報告
- **混合アルカリ系** (N2116含む): Li削減の有望戦略
- **複合水素化物系**: BH₄⁻の回転による協同伝導

### 2.3 N2116の位置づけと課題

N2116 は Li+Na混合系 NASICON 構造で σ ≈ 5×10⁻⁴ S/cm。主要な課題：
- NASICON構造固有のボトルネック（狭いLi+拡散パス）
- 混合アルカリ効果がフレームワークの剛性により制限
- 酸化物アニオンの高電気陰性度がLi+移動度を抑制

## 5. 仮説提案（PICO構造）

### 仮説 H1: 高エントロピーオキシハライド

**Li₂.₅Ti₀.₁₂₅Zr₀.₁₂₅Hf₀.₁₂₅Sn₀.₁₂₅O₀.₅Cl₁.₅Br₁.₀**

| PICO要素 | 内容 |
|----------|------|
| **P** (Population) | 固体電解質材料（Li系、室温動作） |
| **I** (Intervention) | 高エントロピー効果（4元素混合）+ オキシハライドフレームワーク + Cl/Br混合アニオン |
| **C** (Comparison) | N2116 (σ≈5×10⁻⁴), Li₃InCl₆ (σ≈2×10⁻³) |
| **O** (Outcome) | σ > 5×10⁻³ S/cm @ 25°C、電気化学窓 > 4V |

**科学的根拠**:
- Li(Ti,Zr,Hf,Sn)Cl₆ が σ=5.2×10⁻³ S/cm を達成（Luo et al., 2024）
- オキシハライド Li₂.₅O₀.₅Cl₂.₅ が σ=4.5×10⁻³ S/cm（Chen et al., 2024）
- 両者の融合により、エントロピー安定化 + O²⁻によるフレームワーク強化 + 混合ハロゲンによるLi+パス拡大を同時に実現

**相安定性**: Ehull < 30 meV/atom（準安定、高エントロピー項で安定化）

---

### 仮説 H2: フッ素置換アルジロダイト

**Li₆.₅P₀.₅Si₀.₅S₅Cl₀.₅F₀.₅**

| PICO要素 | 内容 |
|----------|------|
| **P** | アルジロダイト構造系固体電解質 |
| **I** | Si⁴⁺/P⁵⁺混合によるLi過剰化 + F⁻/Cl⁻混合アニオンによるパドルホイール促進 |
| **C** | Li₆PS₅Cl (σ≈1.3×10⁻³), Li₆PS₅Cl₀.₅Br₀.₅ (σ≈6.8×10⁻³) |
| **O** | σ > 1×10⁻² S/cm @ 25°C、空気安定性向上 |

**科学的根拠**:
- Li₆PS₅Cl₀.₅Br₀.₅ の混合ハロゲン戦略がパドルホイール回転を促進（Kraft et al., 2018）
- F⁻は最小イオン半径で格子を引き締め、空気安定性を向上
- Si⁴⁺置換で電荷補償として過剰Li⁺を導入、キャリア密度増大
- 活性化エネルギー 0.18 eV（既知アルジロダイトの0.20-0.25 eVより低減予測）

**相安定性**: 既存アルジロダイトとの等構造性から安定と予測

---

### 仮説 H3: 混合アルカリ・アンチペロブスカイト

**Li₂.₅Na₀.₅OCl₀.₇Br₀.₃**

| PICO要素 | 内容 |
|----------|------|
| **P** | アンチペロブスカイト構造（Li-Na混合系） |
| **I** | N2116のLi-Na混合概念をアンチペロブスカイトに転写 + Cl/Br混合 + Li過剰組成 |
| **C** | N2116 (σ≈5×10⁻⁴), Li₃OCl (σ≈8.5×10⁻⁴), Li₂OHCl (σ≈1×10⁻³) |
| **O** | σ > 3×10⁻³ S/cm @ 25°C、Li使用量50%削減 |

**科学的根拠**:
- N2116 の成功要因である混合アルカリフラストレーション効果を活用
- アンチペロブスカイトは本質的にLi+3D伝導パスを持つ
- Na⁺/Li⁺のイオン半径差（1.02Å vs 0.76Å）が格子歪み・disorder を生成
- Cl⁻/Br⁻混合がさらに局所歪みを導入し、協同回転と相乗効果

**相安定性**: 混合アルカリ効果による動的安定化（エントロピー寄与）

## 8. ベイズ最適化プロトコル

### 8.1 アーキテクチャ

```
LHS初期設計(20点) → 測定 → 移行判定 → BO逐次探索(最大30点) → 最適組成同定
                                ↑                        ↓
                          追加LHS(不合格時)          停止判定
```

### 8.2 LHS → BO 移行基準

| 基準 | 閾値 | 根拠 |
|------|------|------|
| データ数 | ≥ 20点 | GP の最低学習データ |
| 変動係数 CV | > 0.3 | 応答面の十分な情報量 |
| 最大伝導度 | > 10⁻⁴ S/cm | 有望な組成空間の確認 |

### 8.3 獲得関数

**ハイブリッド戦略**（EI 70% + Thompson Sampling 30%）：
- **Expected Improvement** (EI): 搾取と探索のバランス（ξ=0.1）
- **Thompson Sampling** (TS): GP事後分布からのサンプリングによる純探索

TS混合比30%は、組成空間の多峰性を考慮した設定。探索初期はTS比率を高め（50%）、収束時に低下（10%）させるアダプティブ戦略も実装済み。

### 8.4 停止基準

| 基準 | 条件 | 優先度 |
|------|------|--------|
| 目標達成 | σ > 5×10⁻³ S/cm を3点以上 | 最優先 |
| 予算限界 | 総50点実行 | 必須 |
| EI収束 | 5回連続 EI < 0.01 | 補助 |

### 8.5 ハイパーパラメータ推定

PyMC によるMCMCサンプリングでGPカーネルハイパーパラメータ（長さスケール、信号分散、ノイズ分散）を逐次更新。10点追加ごとにフルベイズ更新を実施。

## 10. 結論と推奨事項

### 10.1 最優先候補

**仮説 H2（Li₆.₅P₀.₅Si₀.₅S₅Cl₀.₅F₀.₅）** を最優先で実験検証すべきと推奨する。理由：
1. アルジロダイトは既に高伝導度が実証された構造（成功確率が高い）
2. F⁻導入による空気安定性向上は実用面で極めて重要
3. 予測伝導度 1.2×10⁻² S/cm はN2116の24倍

### 10.2 N2116を超えるための設計原理

本分析から導出された5つの設計原理：

1. **混合アニオン戦略**: 単一アニオン系より混合系（Cl⁻/Br⁻/F⁻/O²⁻）が有利
2. **フラストレーション設計**: サイズ不整合イオンの導入による局所歪み
3. **高エントロピー安定化**: 4元素以上の多成分混合で準安定相を動的安定化
4. **Li過剰化**: 電荷補償による過剰Li⁺導入でキャリア密度を増大
5. **構造-メカニズム間ギャップの探索**: 未知の組み合わせに高価値候補が存在

### 10.3 次のステップ

1. H1, H2, H3 の DFT計算による安定性スクリーニング（1-2週間）
2. LHS 20点の合成・評価（4-6週間）
3. BO プロトコルによる逐次最適化（6-12週間）
4. 最良候補の全固体電池セル評価
