# SPReAD プログラム制約（正規ポリシー）

> このファイルは SPReAD（AI for Science 萌芽的挑戦研究創出事業）のプログラム制約の
> **唯一の正規ソース** です。他のスキルファイルでこれらの定数を再定義せず、
> 本ファイルを参照してください。公募要領が更新された場合、本ファイルのみ修正します。

## 事業概要

| 項目 | 内容 |
|------|------|
| 正式名称 | AI for Science 萌芽的挑戦研究創出事業（SPReAD） |
| 主管 | 文部科学省 |
| 実施機関 | 国立研究開発法人 科学技術振興機構（JST） |
| 目的 | あらゆる分野の研究者による AI 利活用を通じた科学研究の高度化・加速化 |

## 予算制約

| 項目 | 金額 |
|------|------|
| 直接経費上限 | 500万円（1課題あたり） |
| 間接経費 | 直接経費の30%（最大150万円） |
| 総額上限 | 650万円（直接経費 + 間接経費） |

### 費目構成（参考）

| 費目 | 説明 | 注意事項 |
|------|------|---------|
| 設備備品費 | ソフトウェアライセンス等 | 汎用PC等は対象外 |
| 消耗品費 | クラウド計算費用（Azure等） | 研究期間内の利用に限定 |
| 旅費 | 国内外の研究打合せ・学会参加 | |
| 人件費・謝金 | RA・技術補佐員等 | |
| その他 | 論文投稿料・データ購入費等 | |

## 期間制約

| 項目 | 内容 |
|------|------|
| 研究期間 | 約180日間 |
| 公募期間 | 令和8年4月17日〜5月18日正午 |
| 選考方法 | 書面審査 + AI インタビュー |

## 審査観点（6基準）

| # | 審査観点 | 重み | 説明 |
|---|---------|------|------|
| 1 | 新規性・革新性 | 高 | AI活用による研究手法の新規性、従来手法との差別化 |
| 2 | 研究の実現可能性 | 高 | 180日間・500万円での実現性、技術的妥当性 |
| 3 | AI活用の適切性 | 高 | AI/MLが研究課題に対して本質的に有効であること |
| 4 | 学術的・社会的インパクト | 中 | 研究成果の波及効果、科学的貢献 |
| 5 | 研究計画の具体性 | 中 | マイルストーン、データ・計算リソース計画の明確さ |
| 6 | 研究者の適格性 | 低 | 研究実績、AI活用能力、実行体制 |

## 応募資格

- 日本国内の大学・研究機関に所属する研究者
- 博士課程学生も応募可（指導教員の承認要）
- 重複制限あり（他の JST 事業との併願制限を確認のこと）

## AI インタビュー

- 採択候補者に対して AI による追加質問（オンライン）
- 研究計画の詳細説明、技術的質問への回答
- 評価結果は書面審査と統合して最終判定

## 成果報告義務

| 報告種別 | 時期 | 内容 |
|---------|------|------|
| 中間報告 | 研究期間の中間時点 | 進捗状況、計画変更の有無 |
| 最終報告 | 研究期間終了後 | 研究成果、AI活用の成果、予算執行実績 |
| 論文謝辞 | 論文発表時 | JST SPReAD への謝辞記載（必須） |
